GGR472 Lab 2
-
This repo contains code for a webpage with external links and one internal link as required by Lab 2.

Link: https://rohanh03.github.io/GGR472-lab-2/
-
Repo Table of Contents:

/data: Data file containing four 'FeatureCollection' GeoJSON files of routes between landmarks. Each 'FeatureCollection' consists of 'LineString' geometries

index.html: HTML file to render webmap with Mapbox GL JS and CSS connections.

script.js: JavaScript file to (1) Visualize a map, (2) Add data from Mapbox GL data sources and GeoJSON files for landmark points, landmark labels, and route lines, (3) Visualize data as layers, one layer for landmark points, one layer for landmark labels, and four layers for four routes between landmarks.

style.css: CSS file to create mapbox container.
